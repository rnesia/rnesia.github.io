Instalasi R pada Ubuntu LTS 20.04

Posted on July 2, 2020 by abdur / 0 Comment

Paket perangkat lunak R yang siap untuk di-install di komputer kita disediakan dan diperbarui oleh R Core Development Team yang tergabung dalam sebuah organisasi nirlaba bernama R Foundation. Installer untuk R tersedia di CRAN (Comprehensive R Archive Network).

Linux Ubuntu merupakan sebuah sistem operasi yang diterbitkan oleh sebuah perusahaan bernama Canonical. Tidak seperti sistem operasi Windows yang haruskita beli dari Microsoft dengan harga mahal, Linux Ubuntu dapat kita unduh, install, dan gunakan secara gratis.

Saya menggunakan Ubuntu versi 20.04 yang dirilis pada bulan April 2020. Versi ini merupakan versi Long-Term Support (LTS) yang berarti akan terus didukung pemeliharaannya oleh Canonical dalam jangka waktu lama, yaitu 5 tahun untuk maintenance updates yang gratis dan 5 tahun berikutnya untuk Extended Security Maintenance yang berbayar. Kabar baiknya, Canonical menyediakan Extended Security Maintenance secara gratis dengan syarat tertentu dalam batas tertentu untuk Ubuntu LTS 14.04 yang sudah memasuki periode 5 tahun kedua (selengkapnya bisa dilihat di sini). Berita ini menunjukkan adanya kemungkinan yang sama untuk versi setelah LTS 14.04.

Dalam artikel ini saya akan menjelaskan langkah-langkah untuk instalasi R versi 4.0.2 pada Linux Ubuntu 20.04. Pastikan selama proses instalasi berlangsung, komputer Anda terhubung dengan internet.

#1 : Buka Terminal

Terminal dapat dibuka dari menu atau dengan menekan ketiga tombol Ctrl Alt T secara bersamaan. Pastikan kita memiliki akses sebagai administrator di komputer yang digunakan karena akan diminta memasukkan password.

#2 : Masukkan Kunci

Kunci (key) merupakan tanda keaslian paket R yang diunduh dari CRAN. Kunci untuk pemasangan paket R pada Ubuntu ini bisa didapatkan dengan menyalin teks berikut (setelah tanda dolar $) ke Terminal, lalu tekan Enter.

Tips : Untuk copy teks berikut, kita dapat menggunakan Ctrl+C, tapi untuk paste pada Terminal, gunakan klik kanan lalu Paste, bukan Ctrl+V.

sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys E298A3A825C0D65DFD57CBB651716619E084DAB9

#3 Siapkan Jalur Unduh

Jalankan kode berikut ini pada Terminal:

sudo apt install gedit

kemudian

sudo gedit /etc/apt/sources.list

Akan muncul halaman file sources.list yang berisi teks seperti pada gambar di bawah ini. File ini terletak pada folder /etc/apt.

Untuk instalasi R versi 4.0.2, salin (copy) lalu tempel (paste) teks berikut ini ke dalam halaman sources.list tersebut. Bisa ditempel di mana saja, misalnya di baris paling bawah. Setelah teks itu disalin, klik Save, lalu tutup file sources.list itu.

deb https://cloud.r-project.org/bin/linux/ubuntu focal-cran40/

Setelah itu, pada Terminal jalankan kode berikut:

sudo apt-get update

Pada tahap ini kita sudah menyambungkan komputer kita dengan repository tempat installer R berada di CRAN sehingga komputer kita sudah siap untuk instalasi R.

#4 Mulai Instalasi

Setelah update selesai, mulai instalasi dengan menjalankan kode berikut:

sudo apt-get install r-base

Saat muncul pertanyaan :

Do you want to continue? [Y/n]

Ketik : 

y

lalu tekan Enter.

Kita akan melihat progress instalasi. Biasanya ini akan berlangsung beberapa menit.

Ketika sudah 100% tercapai, proses ini akan berhenti. Pada tahap ini instalasi R sudah selesai.

#5 Cek Keberhasilan Instalasi

Untuk mengecek apakah instalasi berhasil, dan apakah versi R yang terpasang sudah sesuai dengan target, pada terminal jalankan (setelah tanda dolar $):

R

Bila instalasi berhasil, akan muncul tampilan:

R version 4.0.2 (2020-06-22) -- "Taking Off Again"
Copyright (C) 2020 The R Foundation for Statistical Computing
Platform: x86_64-pc-linux-gnu (64-bit)

R is free software and comes with ABSOLUTELY NO WARRANTY.
You are welcome to redistribute it under certain conditions.
Type 'license()' or 'licence()' for distribution details.

  Natural language support but running in an English locale

R is a collaborative project with many contributors.
Type 'contributors()' for more information and
'citation()' on how to cite R or R packages in publications.

Type 'demo()' for some demos, 'help()' for on-line help, or
'help.start()' for an HTML browser interface to help.
Type 'q()' to quit R.

Kita kemudian bisa mencoba menjalankan kode di R dengan menjalankan kode:

Misalkan kita ingin menampilkan “Assalamualaikum”. Bisa Anda jalankan:

> print("Assalamualaikum")

Akan muncul

[1] "Assalamualaikum"

Anda juga bisa menampilkan gambar hasil analisis data. Sebagai contoh, Anda ingin menampilkan histogram dari 1000 bilangan acak yang terdistribusi Normal. Bisa Anda jalankan kode berikut.

> hist(rnorm(1000))

Akan muncul gambar histogram:

Untuk menutup sesi di R, jalankan kode:

>q()

Akan muncul pertanyaan:

Save workspace image? [y/n/c]:

Kalau Anda ingin menyimpan semua yang sudah dihasilkan pada sesi sebelumnya, ketik y.

Kalau Anda tidak ingin menyimpannya, ketik n.

Kalau Anda tidak jadi menutup sesi, dan ingin melanjutkan sesi di R, ketik c.

Demikian semoga bermanfaat.
